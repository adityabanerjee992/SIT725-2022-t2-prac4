# SIT725-t2-prac3
This will demonstrate the basics of Bootstrapping Express and it will serve a basic HTML page

### Pre-requisite
* Node Js

### Run The Application
Open the project in your own environment

```
$ npm install
$ npm run start

```
Go to the localhost:3000 to open the application.
